# Actividad-formativa-2.-Desarrollo-de-una-Aplicaci-n-Simple-en-L-nea-
El objetivo de esta actividad es desarrollar una aplicación simple en Java o PHP que incluya la declaración de clases, instanciación de objetos, y uso de métodos y constructores. El código se compartirá en un repositorio en línea (como GitHub) y se presentará en una sesión en línea.
